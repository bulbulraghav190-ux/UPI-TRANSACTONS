# 💳 Banking Transactions Dashboard

An interactive **Power BI dashboard** built to analyze banking transactions and customer activity using dynamic visualizations and filters.

## 📊 Features
- Monthly Transaction Analysis
- Amount & Remaining Balance Tracking
- City-wise Comparison
- Bank Sent vs Bank Received
- Payment Mode Analysis
- Merchant Analysis
- Interactive Slicers

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Excel
